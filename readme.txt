Year 2004-2006 Project

当前源代码为2006版本。

C# ASP.Net(.Net 1.1 or later version) SQL2005  

配置后可执行. 完整源代码.

针对当时流行的blogcn， 中国博客网个人做的一个统计系统（来访者记录）。
因为用了黑科技可以获得来访者在blogcn的用户名称， 每个页面停留时间， 等等丰富功能， 而被众多人使用。（数据库十几万保底）
数据库还是用的mdb， 也就是ACCESS， 记录数几千万不止。（屌不屌？）

甚至一度被blogcn认为是一个团队成果， 盗窃其客户信息， 而想着禁止我的域名名称出现在其自定义模板里。
当年劳资刚成年， 血气方刚， 所以， 果断不给它继续免费服务了。
事实证明， blogcn也只是拉一波投资跑路罢了。
